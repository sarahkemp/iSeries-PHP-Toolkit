General
----------------------------
The open-source PHP Toolkit has been developed by IBM and Zend to help you extend PHP code to access IBM i resources. 
The Toolkit's "XML Service" back-end service, written in RPG and using a choice of transports, including ibm_db2 and odbc extensions, is designed to work either statelessly or statefully. 
Its stateful abilities enable developers to call RPG/COBOL/CL programs and APIs while retaining cursors, library lists, the QTEMP library, and more, to enable reuse of existing programs for web applications.

Installation Instructions
----------------------------
For detailed instructions on installing the PHP XMLSERVICE Toolkit, go to: 
[http://files.zend.com/Zend-Server-IBMi/zend-server.htm#i5_sample_scripts.htm](http://files.zend.com/Zend-Server-IBMi/zend-server.htm#i5_sample_scripts.htm)

System Requirements
----------------------------
- IBM: OS 6.1 and above
- Windows: XP and above
- Installed Zend Server on IBMi 

Dependencies
----------------------------
- Enabled ODBC extension. 

To enable ODBC:

1. Go to the Configurations | PHP page.
2. Select the extension, and click Enable in the Action bar. 
3. Click Save to apply changes to Web server.
